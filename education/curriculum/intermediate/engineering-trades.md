# Intermediate Band (Grades 6-8, Ages 11-14) — Engineering & Trades

## Overview
Engineering and trades at the intermediate level moves from craft projects to functional construction. Students learn structural principles, metalworking, plumbing, electrical fundamentals, and construction planning. By Grade 8, students should be able to design and build simple structures and understand renewable energy systems.

## Grade 6: Woodworking and Structural Principles

### Unit 1: Woodworking Projects (Weeks 1-10)
**Duration**: 10 weeks
**Materials**: Hand tools (saws, hammers, chisels, planes, drills), wood, nails, screws, measuring tools, workbench
**Survival Relevance**: Wood is the most available construction material in most environments. Woodworking skills produce furniture, storage, tools, shelter components, and repair capabilities.

#### Learning Objectives
- Use all basic hand tools safely and effectively
- Measure, cut, and join wood using 3+ joint types (butt, lap, mortise-and-tenon, dado)
- Build functional items (shelf, bench, storage box, tool rack)
- Read and create simple woodworking plans

#### Teacher Notes
Safety is paramount. Establish clear workshop rules: no running, no loose clothing near tools, always cut away from your body, secure workpiece before cutting, eye protection when chiseling or drilling. Every student must demonstrate safe tool use before independent work.

#### Lessons
**Lesson 1-2: Tool Review and Safety**
- Activity: Review all hand tools — name, purpose, safe use, maintenance. Demonstrate sharpening (chisels, planes, saws). "A sharp tool is a safe tool — dull tools require more force and slip more easily."
- Key Concepts: Tool maintenance extends tool life, correct technique prevents injury, always wear eye protection when chiseling
- Practice: Demonstrate safe use of every tool in the workshop

**Lesson 3-4: Measuring and Marking**
- Activity: "Measure twice, cut once." Practice measuring to 1mm accuracy. Marking: use a marking gauge, square, and pencil. Transfer measurements accurately. Discuss: why does accuracy matter? (Parts that don't fit waste material and time.)
- Key Concepts: Accuracy in measurement determines project quality, use the same measuring tool throughout a project (tools may differ slightly)
- Practice: Measure and mark 10 cuts with 1mm accuracy

**Lesson 5-6: Joinery**
- Activity: Practice joint types. Butt joint (simplest — two surfaces pressed together + nails/screws). Lap joint (half the thickness removed from each piece — stronger). Mortise-and-tenon (hole + peg — strongest traditional joint). Dado (groove cut to receive a shelf).
- Key Concepts: Joint selection depends on strength requirement and skill level, good joints are the difference between furniture that lasts and furniture that falls apart
- Practice: Create one example of each joint type

**Lesson 7-8: Functional Project**
- Activity: Build a functional item — storage shelf, tool rack, bench, or box. Read the plan. Calculate materials needed. Measure, cut, join, and finish. The item must work.
- Key Concepts: Projects combine all skills — planning, measuring, cutting, joining, finishing, functionality is the test
- Practice: Complete a functional project from plan to finished item

**Lesson 9-10: Design Your Own**
- Activity: Design and build an item the community needs. Create the plan (dimensions, materials list, joint specifications). Get plan approved. Build it. Deliver to the community.
- Key Concepts: Design thinking — identify a need, create a solution, execute, deliver, design for the user not yourself
- Practice: Complete a community-service woodworking project

#### Assessment
- Demonstrate safe use of all basic hand tools
- Create 3 joint types with acceptable fit
- Complete a functional woodworking project from plan to finished item

### Unit 2: Technical Drawing (Weeks 11-16)
**Duration**: 6 weeks
**Materials**: Paper, ruler, compass, protractor, pencils, eraser
**Survival Relevance**: Technical drawings communicate construction plans clearly — without them, builders make mistakes, waste materials, and produce structures that don't fit together

#### Learning Objectives
- Create orthographic drawings (front, top, side views)
- Draw to scale with dimensions
- Create simple isometric (3D-appearing) drawings
- Read and interpret technical drawings made by others

#### Lessons
**Lesson 1-2: Orthographic Views**
- Activity: Place a simple object (box with a notch cut out). Draw it from the front, the top, and the right side — three separate views that together fully describe the shape. "These three views tell a builder everything they need to know."
- Key Concepts: Orthographic projection shows 3D objects in 2D, each view shows one perspective, together they are complete
- Practice: Draw 3-view orthographic projections of 5 objects

**Lesson 3-4: Scale and Dimensions**
- Activity: Draw a project to scale — 1:5 for small items, 1:20 for furniture, 1:50 for structures. Add dimension lines with measurements. Include a title block (project name, scale, date, drawn by).
- Key Concepts: Scale drawings allow large objects to fit on paper, dimensions tell the builder exact sizes, a drawing without dimensions is incomplete
- Practice: Create a scaled, dimensioned drawing of a woodworking project

**Lesson 5-6: Isometric Drawing**
- Activity: Draw on isometric grid — 30° angles create a 3D-appearing image. This helps others visualize what the finished product will look like. Combine with orthographic views for a complete drawing set.
- Key Concepts: Isometric drawings show what the finished item looks like, they complement orthographic views, together they provide complete information
- Practice: Create an isometric drawing of a project and combine with orthographic views

#### Assessment
- Create 3-view orthographic projection of an object
- Draw a project to scale with correct dimensions
- Create an isometric drawing that matches orthographic views

### Unit 3: Structural Principles (Weeks 17-24)
**Duration**: 8 weeks
**Materials**: Popsicle sticks, straws, string, cardboard, weights for testing, rulers
**Survival Relevance**: Understanding structural principles prevents building collapse, optimizes material use, and enables design of shelters, bridges, storage, and other structures

#### Learning Objectives
- Understand load types (compression, tension, shear, torsion)
- Explain why triangles are the strongest structural shape
- Design and test structures for load bearing
- Calculate simple load distributions

#### Lessons
**Lesson 1-2: Forces on Structures**
- Activity: Demonstrate with a cardboard box: push down (compression), pull apart (tension), push sideways (shear), twist (torsion). Every structure experiences some combination of these forces.
- Key Concepts: Structures must resist all forces they encounter, different materials resist different forces better (wood resists tension and compression, stone resists compression)
- Practice: Identify the forces acting on 5 real structures

**Lesson 3-4: The Triangle**
- Activity: Build a square from sticks — push on a corner and it collapses (parallelogram deformation). Add a diagonal — it becomes rigid. "A triangle cannot be deformed without breaking a member. This is why trusses, bridges, and roof structures use triangles."
- Key Concepts: Triangulation is the fundamental structural strategy, it converts compression and tension into stable forces
- Practice: Build a bridge from sticks using triangulation, test with weights

**Lesson 5-6: Structural Testing**
- Activity: Design challenge — build the strongest bridge using limited materials (50 popsicle sticks, glue). Test to failure. Analyze: where did it fail? What would make it stronger? Redesign and retest.
- Key Concepts: Testing reveals design weaknesses, failure analysis improves the next design, engineering is iterative
- Practice: Design, build, test, analyze, and redesign a structure

**Lesson 7-8: Load Distribution**
- Activity: A 100 kg load on a beam supported at two ends — each support carries 50 kg. If the load moves off-center, supports carry unequal loads. Calculate simple load distributions for beams and basic structures.
- Key Concepts: Loads must be transferred to the ground through the structure, uneven loads create stress, foundations must handle the total load
- Practice: Calculate load distribution for 5 structural scenarios

#### Assessment
- Identify forces acting on structures
- Explain why triangulation provides structural strength
- Design and test a structure, analyze failure, and improve
- Calculate simple load distributions

## Grade 7: Metalwork, Plumbing, and Masonry

### Unit 4: Basic Metalworking (Weeks 1-8)
**Duration**: 8 weeks
**Materials**: Metal files, hacksaw, vise, anvil (or heavy steel plate), hammer, tongs, heat source (forge or heavy torch), sharpening stones
**Survival Relevance**: Metal tools, fasteners, and components are essential for all other trades. Metalworking skills enable repair, maintenance, and creation of critical tools.

#### Learning Objectives
- Identify common metals and their properties (iron/steel, copper, aluminum, brass)
- Cut, file, and shape metal using hand tools
- Understand forge theory (heating metal makes it malleable)
- Sharpen and maintain edged tools (knives, axes, chisels)

#### Teacher Notes
Metalworking involves heat, sharp edges, and heavy objects. Safety rules must be absolute: long sleeves, eye protection, leather gloves when handling hot metal, never leave hot metal unattended, always announce when moving hot material.

#### Lessons
**Lesson 1-2: Metal Identification**
- Activity: Identify metals by appearance, weight, magnetism, and spark test (grind on a stone — steel sparks white, iron sparks dull orange, aluminum doesn't spark). Discuss properties: steel is strong and can be hardened, copper conducts heat/electricity, aluminum is light.
- Key Concepts: Different metals suit different purposes, identification enables salvage of useful metals, steel is the most versatile structural metal
- Practice: Identify 10 metal samples by type

**Lesson 3-4: Cutting and Filing**
- Activity: Mark metal with a scriber. Cut with hacksaw (correct blade selection, even strokes, let the saw do the work). File to final shape (flat file for surfaces, round file for holes, bastard cut for rough removal, smooth for finishing).
- Key Concepts: Mark before cutting, measure before marking, filing controls the final dimension, patience produces precision
- Practice: Cut and file 3 metal pieces to specified dimensions (within 1mm)

**Lesson 5-6: Forge Basics**
- Activity: Build a simple forge (fire pit + air supply from bellows or blower). Heat steel to cherry red. Hammer to shape on the anvil. Quench in water to harden, temper by gentle reheating to desired hardness.
- Key Concepts: Steel changes properties with heat treatment, forging reshapes metal, heat treatment determines hardness vs toughness tradeoff
- Practice: Forge a simple item (chisel, hook, or bracket) from steel bar stock

**Lesson 7-8: Tool Sharpening and Maintenance**
- Activity: Sharpen knives, axes, and chisels. Coarse stone for setting the edge angle, medium stone for refining, fine stone for polishing. Strop for final edge. Maintain tools: oil to prevent rust, check handles, tighten heads.
- Key Concepts: A sharp edge is geometry — consistent angle is more important than force, regular maintenance is cheaper than replacement
- Practice: Sharpen 3 different edged tools to working sharpness

#### Assessment
- Identify 5+ metals by their properties
- Cut and file metal to specified dimensions
- Forge a simple item from heated steel
- Sharpen edged tools to working sharpness

### Unit 5: Plumbing Concepts (Weeks 9-16)
**Duration**: 8 weeks
**Materials**: Pipe (PVC, copper, or salvaged), fittings, sealant/tape, measuring tools, water source for testing
**Survival Relevance**: Water delivery and waste removal are critical infrastructure. Plumbing skills maintain the systems that keep communities healthy.

#### Learning Objectives
- Understand water system components (source, storage, treatment, distribution, drainage)
- Cut, join, and seal pipe connections
- Design a gravity-fed water distribution system
- Troubleshoot common plumbing problems (leaks, blockages, pressure issues)

#### Lessons
**Lesson 1-2: Water System Overview**
- Activity: Trace the community's water system from source to tap to drain. Source (well, spring, stream, rainwater) → Storage (tank, cistern) → Treatment (filter, boil, chemical) → Distribution (pipes, taps) → Drainage (gray water to garden, black water to latrine).
- Key Concepts: Every component matters — failure at any point disrupts the whole system, gravity is the cheapest and most reliable pressure source
- Practice: Map the community's complete water system

**Lesson 3-4: Pipe Work**
- Activity: Practice cutting pipe (PVC: saw + deburr; copper: tube cutter). Join PVC (primer + cement). Join copper (solder or compression fittings). Test for leaks (pressurize and inspect every joint).
- Key Concepts: Clean cuts and proper joining prevent leaks, test before burying or covering pipes, leak prevention is easier than leak repair
- Practice: Create a leak-free pipe assembly with at least 5 joints

**Lesson 5-6: Gravity-Fed System Design**
- Activity: Design a gravity-fed water system: storage tank on high ground → pipe to distribution points → taps at use points. Calculate: head (height difference) determines pressure. 1 meter of head = ~0.1 bar ≈ 1.4 psi. Size pipe for required flow rate.
- Key Concepts: Gravity systems require no pumps or electricity, head height determines pressure and flow, pipe size affects flow rate
- Practice: Design a gravity-fed system for a building with calculations for pressure and flow

**Lesson 7-8: Troubleshooting**
- Activity: Common problems: leaky joints (tighten or reseal), low pressure (check for blockages, leaks, or insufficient head), blockages (plunge, rod, or disassemble), frozen pipes (insulate, heat gently). Practice diagnosing and fixing each.
- Key Concepts: Systematic troubleshooting: start at the source and work downstream, most problems have simple causes
- Practice: Diagnose and fix 5 simulated plumbing problems

#### Assessment
- Describe all components of a water system
- Create a leak-free pipe assembly
- Design a gravity-fed water system with correct calculations
- Diagnose and fix common plumbing problems

### Unit 6: Masonry (Weeks 17-24)
**Duration**: 8 weeks
**Materials**: Bricks or stones, mortar (cement + sand + water, or lime + sand + water), trowel, level, string line, measuring tools
**Survival Relevance**: Masonry creates permanent structures — walls, foundations, ovens, chimneys, cisterns, and defensive works that last generations

#### Learning Objectives
- Mix mortar to correct consistency
- Lay brick and stone in straight, level courses
- Build structural corners and piers
- Understand foundation requirements for masonry walls

#### Lessons
**Lesson 1-2: Mortar Mixing**
- Activity: Standard mortar: 1 part cement to 3 parts sand + water to workable consistency (like thick peanut butter). Lime mortar alternative: 1 part lime to 3 parts sand + water. Mix thoroughly. Test: should hold its shape when squeezed but not crumble.
- Key Concepts: Mortar ratio determines strength, too much water weakens mortar, mixing must be thorough, mortar has a working time (use within 2 hours)
- Practice: Mix 3 batches of mortar and test consistency

**Lesson 3-4: Brick Laying**
- Activity: Set a string line for level. Apply mortar to foundation and brick bottom. Place brick. Check level and plumb (vertical). Tap to adjust. Next brick — butter the end and press against the previous brick. Keep courses level.
- Key Concepts: Level and plumb are non-negotiable, consistent mortar joints look better and are stronger, running bond (offset each course by half a brick) prevents vertical weak lines
- Practice: Lay 3 courses of a straight brick wall

**Lesson 5-6: Corners and Structural Elements**
- Activity: Corners are the reference points — build corners first, then fill in between using a string line. Practice corner construction with interlocking courses. Build a pier (column) using stacked courses.
- Key Concepts: If the corners are right, the wall will be right, corners require careful planning and execution
- Practice: Build a corner column 5 courses high

**Lesson 7-8: Stone Masonry**
- Activity: Lay stone wall using available natural stone. Select stones for face and fit. Fill gaps with smaller stones and mortar. "Stone masonry is puzzle-solving — find the right stone for each spot."
- Key Concepts: Stone masonry uses irregular materials — skill is in selection and placement, a well-built stone wall can last centuries, through-stones (spanning the full wall width) tie the wall together
- Practice: Build a section of stone wall 1 meter long and 50 cm high

#### Assessment
- Mix mortar to correct consistency
- Lay 3 courses of level, plumb brickwork
- Build a structural corner
- Complete a section of stone wall

## Grade 8: Electrical, Mechanical, and Construction

### Unit 7: Electrical Fundamentals (Weeks 1-10)
**Duration**: 10 weeks
**Materials**: Wire, batteries, bulbs, switches, resistors, multimeter (if available), motors, generators
**Survival Relevance**: Electrical knowledge enables lighting, radio communication, power tool use, medical equipment operation, and renewable energy system maintenance

#### Learning Objectives
- Understand voltage, current, and resistance (V = IR, Ohm's Law)
- Build and analyze series and parallel circuits
- Use a multimeter to measure voltage, current, and resistance
- Wire simple circuits safely (proper connections, fusing, grounding)

#### Lessons
**Lesson 1-2: Voltage, Current, Resistance**
- Activity: Water analogy — voltage is water pressure (force pushing current), current is water flow (amount of electricity moving), resistance is pipe diameter (opposition to flow). V = IR. If voltage doubles, current doubles. If resistance doubles, current halves.
- Key Concepts: Ohm's Law governs all circuits, understanding it enables diagnosis and design, units: Volts (V), Amps (A), Ohms (Ω)
- Practice: Calculate V, I, or R for 15 problems using Ohm's Law

**Lesson 3-4: Series and Parallel Circuits**
- Activity: Build a series circuit — components in a line. Current is the same through all components. Total resistance adds. If one component fails, all fail. Build a parallel circuit — components on separate branches. Voltage is the same across all branches. Total resistance decreases. If one fails, others continue.
- Key Concepts: Series for simplicity, parallel for reliability, most real circuits combine both
- Practice: Build, draw, and analyze 5 series and 5 parallel circuits

**Lesson 5-6: Multimeter Use**
- Activity: Learn to use a multimeter: set to voltage range → measure across a component. Set to current range → measure in-line. Set to resistance range → measure disconnected component. Practice measuring real circuits.
- Key Concepts: A multimeter is the most important electrical diagnostic tool, always start at the highest range and work down, never measure resistance on a live circuit
- Practice: Measure V, I, and R in 5 different circuits and verify Ohm's Law

**Lesson 7-8: Practical Wiring**
- Activity: Wire a simple lighting circuit: battery/power source → switch → fuse → light → return. Discuss: why fuses? (Prevent fire from overcurrent.) Why switches? (Control.) Why proper connections? (Poor connections cause heat, resistance, fire.)
- Key Concepts: Safety is not optional — improper wiring causes fires, fuses protect circuits, connections must be mechanically secure and electrically sound
- Practice: Wire a complete lighting circuit with switch and fuse that works safely

**Lesson 9-10: Motors and Generators**
- Activity: A motor converts electricity to motion. A generator converts motion to electricity. They are the SAME device run in different directions. Demonstrate with a small DC motor — spin it and measure voltage output. Connect to a light.
- Key Concepts: Electromagnetism links electricity and motion, generators are the source of all electricity, understanding this enables building and maintaining power systems
- Practice: Build a simple generator and measure its output

#### Assessment
- Apply Ohm's Law to calculate circuit values
- Build and analyze series and parallel circuits
- Use a multimeter for electrical measurement
- Wire a safe, functional lighting circuit

### Unit 8: Small Engine Mechanics (Weeks 11-16)
**Duration**: 6 weeks
**Materials**: Small engine (lawn mower, chainsaw, generator — even non-functional for study), basic tools, fuel system supplies
**Survival Relevance**: Small engines power generators, water pumps, chainsaws, and vehicles. Understanding their operation enables maintenance and repair of critical community equipment.

#### Learning Objectives
- Explain the four-stroke engine cycle (intake, compression, combustion, exhaust)
- Identify major engine components and their functions
- Perform basic maintenance (oil change, air filter, spark plug, fuel system)
- Diagnose common engine problems (won't start, runs rough, overheats)

#### Lessons
**Lesson 1-2: How Engines Work**
- Activity: Demonstrate the four-stroke cycle with diagrams and a disassembled engine: Intake (air + fuel enter cylinder), Compression (piston compresses mixture), Combustion (spark ignites, expanding gases push piston), Exhaust (burned gases pushed out). This cycle repeats hundreds of times per minute.
- Key Concepts: Internal combustion converts fuel to motion, the four strokes happen in sequence, each stroke has a specific function
- Practice: Identify the four strokes and describe what happens during each

**Lesson 3-4: Engine Components**
- Activity: Disassemble (if available) and identify: cylinder block, piston, connecting rod, crankshaft, valves, spark plug, carburetor, air filter, oil system, cooling system (air or liquid), exhaust.
- Key Concepts: Every component has a specific function, understanding function enables diagnosis, wear on any component affects performance
- Practice: Identify and explain the function of all major components

**Lesson 5-6: Maintenance**
- Activity: Perform basic maintenance: change oil (drain, refill with correct type and amount), replace air filter, inspect and replace spark plug (read the old plug — black = rich, white = lean, tan = correct), clean or adjust carburetor, check fuel system.
- Key Concepts: Regular maintenance prevents breakdown, it's cheaper (in time and parts) than repair, maintenance schedule varies by engine type and use
- Practice: Perform a complete maintenance service on a small engine

#### Assessment
- Explain the four-stroke engine cycle
- Identify and describe all major engine components
- Perform a complete basic maintenance service

### Unit 9: Construction Planning and Renewable Energy (Weeks 17-28)
**Duration**: 12 weeks
**Materials**: Design/drawing materials, construction tools, building materials, solar panel (if available), wind measurement tools
**Survival Relevance**: The ability to plan and execute construction projects — and power them with renewable energy — is fundamental to community infrastructure development

#### Learning Objectives
- Plan a construction project from concept through completion
- Create construction drawings with materials lists and schedules
- Design and build a small structure (shed, shelter, workshop addition)
- Understand renewable energy systems (solar, wind, micro-hydro) at a conceptual level

#### Lessons
**Lesson 1-2: Project Planning**
- Activity: Choose a community construction need (storage shed, workshop addition, animal shelter, observation post). Define: purpose, size, location, materials, tools needed, timeline, labor requirements.
- Key Concepts: Planning prevents waste, scope definition prevents scope creep, realistic timelines include contingency
- Practice: Create a complete project plan for a small structure

**Lesson 3-4: Construction Drawings**
- Activity: Create a full drawing set: site plan (where on the property), floor plan (from above), elevations (each side), cross-section (cut through), foundation detail. Include dimensions, materials specifications, and a materials list with quantities.
- Key Concepts: Complete drawings are the contract between designer and builder, missing information causes mistakes
- Practice: Complete a full drawing set for the project

**Lesson 5-6: Foundation and Framing**
- Activity: Build the foundation (compacted gravel, concrete piers, or stone). Frame the structure (posts, beams, rafters). Connect with appropriate joinery (notch, bolt, strap). Check level and plumb at every stage.
- Key Concepts: Foundation is the most important element — everything sits on it, framing must be square and plumb, structural connections must be strong enough for the loads
- Practice: Participate in building the foundation and frame of the project

**Lesson 7-8: Enclosure and Finishing**
- Activity: Add walls (boards, panels, or masonry), roof (rafters, sheathing, roofing material), door, and any windows. Finish interior as needed. The structure must be weather-tight and functional.
- Key Concepts: Waterproofing prevents rot, ventilation prevents moisture damage, finishing makes the structure usable
- Practice: Complete the enclosure and finishing of the project

**Lesson 9-10: Solar Energy Basics**
- Activity: How solar panels work (photovoltaic effect — light → electricity). System components: panel, charge controller, battery, inverter (for AC). Calculate: a 100W panel in 5 hours of sun produces 500 Wh per day. What can that power?
- Key Concepts: Solar is the most accessible renewable energy, system sizing must match needs, batteries are the most expensive and fragile component
- Practice: Calculate solar system size for a specific application (lighting, radio charging, medical equipment)

**Lesson 11-12: Wind and Micro-Hydro Concepts**
- Activity: Wind energy: kinetic energy of moving air → generator → electricity. Power increases with the CUBE of wind speed (double the speed = 8x the power). Micro-hydro: water flow → turbine → generator → electricity. Power depends on flow rate and head (height drop).
- Key Concepts: Wind and hydro produce power continuously (unlike solar), site assessment is critical (need consistent wind or water flow), maintenance requirements differ
- Practice: Assess the community site for wind and hydro potential

#### Assessment
- Plan a construction project with complete documentation
- Participate in building a functional structure
- Calculate solar system sizing for a specific application
- Assess renewable energy potential for the community site
