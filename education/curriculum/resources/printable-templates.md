# Printable Templates — Worksheets and Reference Materials

## Overview
These templates support instruction across the curriculum. They can be printed if a printer is available, hand-copied onto paper, or drawn on a chalkboard. Each template is described with enough detail to be reproduced by hand.

### Periodic Table of Elements (Intermediate/Advanced)
```
Reproduce a standard periodic table with:
- Atomic number, symbol, name, atomic mass for each element
- Color-coding by category: metals, nonmetals, metalloids, noble gases
- Key properties noted for survival-relevant elements:
  - H (hydrogen): lightest, explosive, water component
  - C (carbon): basis of organic chemistry, charcoal
  - N (nitrogen): fertilizer component, 78% of air
  - O (oxygen): 21% of air, combustion
  - Na (sodium): salt component, preservation
  - Cl (chlorine): water purification, salt component
  - Fe (iron): tools, construction, blood
  - Cu (copper): electrical wire, plumbing, antimicrobial
  - Ag (silver): antimicrobial, trade value
  - Pb (lead): toxic, battery, radiation shielding
```

## Star Charts and Navigation

### Northern Hemisphere Star Chart
```
Reproduce showing:
- Big Dipper → pointer stars → North Star (Polaris)
- Little Dipper (with Polaris at end of handle)
- Cassiopeia (W shape, opposite Big Dipper from Polaris)
- Orion (winter) with Betelgeuse and Rigel labeled
- Summer Triangle: Vega, Deneb, Altair
- Note: Polaris indicates TRUE NORTH
- Note: Altitude of Polaris above horizon = your latitude
```

### Southern Hemisphere Star Chart
```
Reproduce showing:
- Southern Cross (Crux) — 4 bright stars in cross shape
- How to find south: extend long axis of cross 4.5× its length
- Pointer stars (Alpha and Beta Centauri) — confirm direction
- Magellanic Clouds (satellite galaxies visible as fuzzy patches)
- Note: No south pole star — must use Southern Cross method
```
